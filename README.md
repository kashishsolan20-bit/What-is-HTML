# What-is-HTML
HTML (HyperText Markup Language) is used to create the structure of a webpage.
It tells the browser what to display like text, images, buttons, links, etc.

HTML is made using tags.
<h1>Hello World</h1>
# Basic Structure of an HTML Page
<!DOCTYPE html>
<html>
<head>
<title>My Website</title>
</head>

<body>

<h1>Hello</h1>
<p>This is my webpage</p>

</body>
</html>
